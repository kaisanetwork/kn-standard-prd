# Metrics Framework Selector

Pick one primary framework and optionally one supporting framework.

## Quick Selection Guide

- **AARRR**: Best for growth funnels and acquisition-to-revenue motion.
- **HEART**: Best for UX quality and interaction health.
- **North Star**: Best for aligning teams around one core value metric.
- **OKRs**: Best for initiative planning with clear objective/KR tracking.

## Minimum Metric Spec

Every PRD metric should define:

- Name and formula
- Baseline (current value)
- Target (numeric)
- Time window
- Owner
- Source of truth (dashboard/event)

## Recommended Metric Mix

- 1 North Star metric
- 2-4 supporting metrics
- At least 1 leading + 1 lagging indicator

## Examples

### Example A: Onboarding Feature

- North Star: Activation rate
- Supporting: Time-to-value, onboarding completion, D7 retention

### Example B: Checkout Improvement

- North Star: Checkout conversion
- Supporting: Drop-off at payment step, median checkout time, support tickets/order

### Example C: AI Assistant Feature

- North Star: Task success rate
- Supporting: Hallucination rate, human override rate, p95 response latency, cost per successful task
